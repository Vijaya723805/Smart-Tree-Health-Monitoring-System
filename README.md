# Smart Tree Health Monitoring System

## Description
The Smart Tree Health Monitoring System is a web-based application designed to monitor and analyze the health condition of trees. It helps in identifying potential health issues at an early stage and supports effective tree care and management.

## Features
- Monitoring of tree health parameters  
- Tracking of soil moisture, temperature, humidity, and nutrient levels  
- Data processing and analysis using Python  
- Health status visualization through a dashboard  
- Secure data storage using MySQL  
- Simple and user-friendly interface  

## Tech Stack
- Frontend: HTML, CSS  
- Backend: Python  
- Database: MySQL  

## Purpose
This project aims to support sustainable environmental management by providing a digital solution for monitoring and maintaining tree health.
